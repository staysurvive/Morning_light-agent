# 模块一：工作台 (Dashboard)

## 路由

```
/    → Dashboard 首页
```

## 功能描述

工作台是登录后的首页，提供全局运营视角的数据大盘，让管理员一眼掌握平台运行状态。

## 趋势图（中部2个）

### 调用量趋势
- 类型：折线图
- X轴：最近7天日期
- Y轴：调用次数
- 数据点：每天的总调用量

### Token消耗趋势
- 类型：折线图
- X轴：最近7天日期
- Y轴：Token数量
- 可区分 input tokens 和 output tokens
