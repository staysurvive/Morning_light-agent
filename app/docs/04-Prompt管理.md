### Prompt数据结构

```typescript
interface Prompt {
  id: string
  name: string
  description: string
  category: string                // 分类
  tags: string[]                  // 标签
  content: string                 // Prompt内容（支持 {{variable}} 变量）
  variables: PromptVariable[]     // 变量定义
  version: string                 // 当前版本
  status: 'draft' | 'published'   // 状态
  usageCount: number              // 使用次数
  linkedAgentCount: number        // 关联Agent数
  createdBy: string               // 创建人
  createdAt: string
  updatedAt: string
}

interface PromptVariable {
  name: string                    // 变量名
  type: 'string' | 'number' | 'boolean' | 'text'
  description: string             // 变量说明
  defaultValue?: string           // 默认值
  required: boolean               // 是否必填
}
```

## Mock数据要点

### Prompt列表（至少15条）

分类覆盖：
- 客服（4条）：对话模板、投诉处理、售后服务、FAQ回答
- 开发（3条）：代码审查、Bug分析、技术文档生成
- 营销（3条）：文案生成、社媒内容、邮件营销
- 分析（2条）：数据分析、报告生成
- 工具（2条）：翻译助手、摘要提取
- 通用（1条）：通用对话

每条Prompt需要：
- 完整的Prompt内容（至少100字）
- 2-3个变量定义
- 合理的使用次数
- 版本号
