# 模块五：知识库管理 (RAG)

## 路由

```
/knowledge              → 知识库列表
/knowledge/create       → 创建知识库
/knowledge/:id          → 知识库详情
/knowledge/:id/documents → 文档管理
/knowledge/:id/segments  → 分段管理
/knowledge/:id/test      → 检索测试
```

### 知识库数据结构

```typescript
interface KnowledgeBase {
  id: string
  name: string
  description: string
  status: 'ready' | 'indexing' | 'error' | 'empty'
  documentCount: number
  segmentCount: number
  vectorCount: number
  storageSize: string             // "2.3 GB"
  embeddingModel: string          // Embedding模型
  indexConfig: {
    chunkStrategy: 'auto' | 'custom' | 'paragraph' | 'sentence'
    chunkSize: number             // 分段大小(tokens)
    chunkOverlap: number          // 分段重叠(tokens)
    separator: string             // 分隔符
  }
  retrievalConfig: {
    strategy: 'vector' | 'fulltext' | 'hybrid'
    topK: number
    similarityThreshold: number
    rerankModel?: string
  }
  linkedAgentCount: number
  createdBy: string
  createdAt: string
  updatedAt: string
}
```

### 文档数据结构

```typescript
interface Document {
  id: string
  knowledgeBaseId: string
  fileName: string
  fileType: 'pdf' | 'docx' | 'md' | 'txt' | 'html' | 'csv'
  fileSize: string                // "2.3 MB"
  status: 'pending' | 'processing' | 'completed' | 'failed'
  segmentCount: number
  wordCount: number
  errorMessage?: string           // 失败原因
  uploadedBy: string
  uploadedAt: string
  processedAt?: string
}
```

### 文档操作
- 上传：支持拖拽上传，支持批量上传
- 预览：查看文档原始内容
- 重新处理：重新分段和索引
- 删除：删除文档及其所有分段

### 支持的文件类型
- PDF (.pdf)
- Word (.docx)
- Markdown (.md)
- 纯文本 (.txt)
- HTML (.html)
- CSV (.csv)
