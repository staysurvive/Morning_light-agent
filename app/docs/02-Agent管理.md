# 模块二：Agent管理

## 路由

```
/agents                → Agent列表
/agents/create         → 创建Agent
/agents/:id            → Agent详情
/agents/:id/edit       → 编辑Agent
/agents/:id/test       → Agent测试/调试
/agents/:id/versions   → 版本管理
/agents/:id/monitor    → Agent监控
```

### 表格字段

| 字段 | 说明 |
|------|------|
| 名称 | Agent名称 + 简短描述 |
| 类型 | 对话型 / 工具型 / 分析型 / 创作型 / 工作流 |
| 状态 | 运行中(绿) / 已停止(灰) / 异常(红) / 草稿(黄) |
| 关联模型 | 使用的LLM模型名称 |
| 成功率 | 最近7天的成功率 |
| 调用量 | 最近7天的调用次数 |
| 最后运行 | 最后一次执行时间 |
| 创建时间 | Agent创建时间 |
| 操作 | 启动/停止、编辑、测试、删除、更多 |

### 筛选条件
- 关键词搜索（名称、描述）
- 类型筛选（多选）
- 状态筛选（多选）
- 排序（创建时间、调用量、成功率）

### 表单字段详细

#### 基本信息
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | ✅ | Agent名称，2-50字符 |
| description | string | ✅ | Agent描述，最多200字符 |
| type | enum | ✅ | 对话型/工具型/分析型/创作型/工作流 |
| icon | string | ❌ | Agent图标标识 |
| tags | string[] | ❌ | 标签，用于分类 |

#### 模型配置
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| modelId | string | ✅ | 关联的模型ID |
| temperature | number | ✅ | 0-2，默认0.7 |
| maxTokens | number | ✅ | 最大输出Token数 |
| topP | number | ❌ | 0-1，默认1.0 |
| systemPrompt | string | ✅ | 系统提示词，可选择模板或自定义 |
| promptTemplateId | string | ❌ | 关联的Prompt模板ID |

#### 知识库配置
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| ragEnabled | boolean | ✅ | 是否启用RAG |
| knowledgeBaseIds | string[] | ❌ | 关联的知识库ID列表 |
| retrievalStrategy | enum | ❌ | vector/fulltext/hybrid |
| topK | number | ❌ | 检索返回数量，默认5 |
| similarityThreshold | number | ❌ | 相似度阈值，默认0.7 |

#### 工具配置
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| toolsEnabled | boolean | ✅ | 是否启用工具调用 |
| toolIds | string[] | ❌ | 可用工具ID列表 |

#### 高级配置
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| welcomeMessage | string | ❌ | 开场白 |
| suggestedQuestions | string[] | ❌ | 建议问题列表 |
| maxTurns | number | ❌ | 对话轮数限制 |
| timeout | number | ❌ | 超时时间（秒） |

## Mock数据要点

### Agent列表（至少12条）

需要覆盖：
- 不同类型（对话型、工具型、分析型、创作型、工作流）
- 不同状态（运行中、已停止、异常、草稿）
- 不同的成功率范围（85%-99%）
- 不同的调用量级别
- 合理的中文名称和描述

### 版本数据
- 每个Agent 3-5个版本
- 版本间有合理的配置差异

### 监控数据
- 7天的时序数据
- 数据有合理波动
- 包含一些异常点
