### 工具数据结构

```typescript
interface Tool {
  id: string
  name: string
  description: string
  type: 'builtin' | 'http_api' | 'custom_function'
  status: 'enabled' | 'disabled' | 'error'
  config: HttpApiConfig | CustomFunctionConfig
  functionDefinition: FunctionDefinition  // OpenAI Function Calling格式
  permissions: {
    allowedAgentIds: string[] | 'all'
    rateLimit: number                     // 次/分钟
    timeout: number                       // 超时秒数
  }
  stats: {
    callCount7d: number
    successRate: number
    avgLatency: number                    // ms
  }
  linkedAgentCount: number
  createdBy: string
  createdAt: string
  updatedAt: string
}

interface HttpApiConfig {
  method: 'GET' | 'POST' | 'PUT' | 'DELETE'
  url: string
  headers: Record<string, string>
  parameters: ToolParameter[]
  responseMapping: {
    resultField: string
    errorField: string
  }
}

interface ToolParameter {
  name: string
  type: 'string' | 'number' | 'boolean' | 'object' | 'array'
  required: boolean
  description: string
  defaultValue?: string
}

interface FunctionDefinition {
  name: string
  description: string
  parameters: {
    type: 'object'
    properties: Record<string, {
      type: string
      description: string
      enum?: string[]
    }>
    required: string[]
  }
}
```

## Mock数据要点

### 工具列表（至少12个）

| 名称 | 类型 | 状态 | 说明 |
|------|------|------|------|
| 网页搜索 | 内置工具 | 启用 | 搜索互联网获取信息 |
| 代码执行 | 内置工具 | 启用 | 执行Python/JS代码 |
| 图片生成 | 内置工具 | 启用 | 调用DALL-E生成图片 |
| 文件读取 | 内置工具 | 启用 | 读取上传的文件内容 |
| 天气查询 | HTTP API | 启用 | 查询城市天气信息 |
| 汇率转换 | HTTP API | 启用 | 实时汇率查询和转换 |
| 数据库查询 | 自定义函数 | 启用 | 查询业务数据库 |
| 邮件发送 | HTTP API | 启用 | 发送邮件通知 |
| 日程管理 | HTTP API | 禁用 | 管理日程和提醒 |
| 知识图谱 | 自定义函数 | 启用 | 查询知识图谱 |
| OCR识别 | HTTP API | 错误 | 图片文字识别 |
| 语音转文字 | HTTP API | 启用 | 语音识别转文字 |

每个工具需要完整的Function Definition和参数定义。
