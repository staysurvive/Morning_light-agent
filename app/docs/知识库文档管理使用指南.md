# 知识库文档管理使用指南

## 功能概述

辰光Agent平台的知识库文档管理功能提供了完整的文档上传、OCR识别和人工校对流程，帮助您快速构建高质量的RAG知识库。

## 主要功能

### 1. 文档上传

支持多种文档格式：
- **文档类型**: PDF, DOCX, TXT, Markdown
- **图片类型**: PNG, JPG, JPEG

### 2. 图片OCR识别

上传图片后，系统会自动调用OCR模型识别图片中的文字内容，无需手动操作。

**识别流程**：
1. 上传图片文件
2. 系统自动开始OCR识别（显示进度提示）
3. 识别完成后，左侧显示原始图片，右侧显示识别结果
4. 自动进入编辑模式，可以立即校对

### 3. 人工校对

OCR识别可能存在错误，系统提供了便捷的人工校对功能：

**校对步骤**：
1. 在右侧编辑框中查看OCR识别结果
2. 对照左侧原始图片，修正识别错误
3. 点击"保存"按钮保存校对后的内容
4. 系统自动开始文档处理和向量化

**编辑功能**：
- ✅ 大文本框，方便编辑长文本
- ✅ 等宽字体，便于对齐和阅读
- ✅ 支持滚动查看
- ✅ 取消按钮可恢复原始内容

### 4. 文档管理

**左侧文档列表**：
- 显示所有已上传的文档
- 文件名、大小、状态一目了然
- 点击文档查看内容
- 支持删除操作

**文档状态**：
- 🟢 **已完成**: 文档已处理完成，可以被检索
- 🟡 **处理中**: 正在进行分段和向量化
- 🔴 **失败**: 处理失败，需要重新上传
- ⚪ **等待中**: 等待处理

## 使用流程

### 场景1: 上传普通文档

```
1. 进入知识库列表
   ↓
2. 点击知识库名称进入详情
   ↓
3. 点击"上传文档/图片"按钮
   ↓
4. 选择PDF/DOCX/TXT/MD文件
   ↓
5. 系统自动上传并处理
   ↓
6. 在右侧查看文档内容
```

### 场景2: 上传图片并OCR识别

```
1. 进入知识库详情页面
   ↓
2. 点击"上传文档/图片"按钮
   ↓
3. 选择PNG/JPG图片文件
   ↓
4. 系统自动开始OCR识别（2-5秒）
   ↓
5. 左侧显示原始图片，右侧显示识别结果
   ↓
6. 在右侧编辑框中校对内容
   ↓
7. 点击"保存"按钮
   ↓
8. 系统开始处理文档
```

## 常见问题

### Q1: 支持哪些图片格式？
A: 目前支持 PNG, JPG, JPEG 格式。建议使用清晰的扫描件或照片。

### Q2: OCR识别准确率如何？
A: 识别准确率取决于图片质量。建议：
- 使用高分辨率图片
- 确保文字清晰可见
- 避免倾斜和模糊
- 光线充足，对比度高

### Q3: 可以批量上传吗？
A: 当前版本暂不支持批量上传，后续版本会添加此功能。

### Q4: 上传的文档可以修改吗？
A: 目前只支持OCR识别结果的校对。普通文档上传后不可修改，需要重新上传。

### Q5: 文档处理需要多长时间？
A: 取决于文档大小和内容：
- 小文档（<1MB）: 10-30秒
- 中等文档（1-10MB）: 30秒-2分钟
- 大文档（>10MB）: 2-5分钟

### Q6: 删除文档会影响已有的对话吗？
A: 删除文档会从知识库中移除，但不会影响历史对话记录。
