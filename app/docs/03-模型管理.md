### 表格字段

| 字段 | 说明 |
|------|------|
| 模型名称 | 模型显示名称 |
| 模型标识 | API调用时使用的model ID |
| 供应商 | 所属供应商 |
| 能力标签 | Function Call / Vision / 长上下文 / Embedding |
| 上下文长度 | 最大上下文窗口 |
| 状态 | 可用 / 不可用 / 限流中 |
| 输入价格 | 每1K tokens的输入价格 |
| 输出价格 | 每1K tokens的输出价格 |
| 调用量 | 最近7天调用次数 |
| 操作 | 编辑、测试、禁用、删除 |

### 模型数据结构

```typescript
interface Model {
  id: string
  name: string                    // 显示名称
  modelId: string                 // API model ID
  providerId: string              // 供应商ID
  providerName: string            // 供应商名称
  capabilities: ModelCapability[] // 能力列表
  contextLength: number           // 上下文长度
  status: 'available' | 'unavailable' | 'rate_limited'
  inputPrice: number              // 输入价格 (每1K tokens)
  outputPrice: number             // 输出价格 (每1K tokens)
  currency: string                // 货币单位
  callCount7d: number             // 7天调用量
  tokenUsage7d: number            // 7天Token消耗
  isDefault: boolean              // 是否为默认模型
  config: {
    maxTokens: number
    temperature: number
    topP: number
  }
  createdAt: string
  updatedAt: string
}

type ModelCapability = 'function_call' | 'vision' | 'long_context' | 'embedding' | 'streaming'
```

### 供应商数据结构

```typescript
interface ModelProvider {
  id: string
  name: string                    // 供应商名称
  type: 'openai' | 'anthropic' | 'aliyun' | 'azure' | 'local' | 'custom'
  status: 'connected' | 'disconnected' | 'error'
  endpoint: string                // API端点
  apiKeys: ApiKeyConfig[]         // API Key列表（支持多Key负载均衡）
  config: {
    timeout: number               // 超时时间
    maxRetries: number            // 最大重试次数
    proxy?: string                // 代理地址
    orgId?: string                // 组织ID
  }
  modelCount: number              // 关联模型数量
  createdAt: string
  updatedAt: string
}

interface ApiKeyConfig {
  id: string
  key: string                     // API Key（脱敏显示）
  weight: number                  // 负载均衡权重
  status: 'active' | 'expired' | 'rate_limited'
  lastUsed: string
}
```
