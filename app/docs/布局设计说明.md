# 辰光Agent平台 - 布局设计说明

## 设计理念

参考现代化企业级后台管理系统的设计规范，打造专业、美观、易用的用户界面。

## 整体布局

### 左右分栏结构

```
┌─────────────────────────────────────────┐
│  左侧导航栏  │  右侧内容区              │
│  (256px)    │  (自适应)                │
│             │                          │
│  深色背景   │  浅色背景                │
└─────────────────────────────────────────┘
```

## 左侧导航栏

### 1. Logo区域（顶部）

**位置**: 固定在顶部，高度 64px

**设计元素**:
- Logo图标：蓝紫渐变圆角方块
- 产品名称：辰光 Agent
- 版本标识：Enterprise（小字灰色）

**样式**:
```css
背景：深色（slate-900 到 slate-800 渐变）
Logo背景：蓝色到紫色渐变 + 阴影
文字：白色主标题 + 灰色副标题
边框：底部深色边框分隔
```

### 2. 导航菜单（中间）

**分组结构**:
1. **平台功能**（主要功能模块）
   - 工作台
   - Agent管理
   - 模型管理
   - Prompt管理
   - 知识库
   - 工具管理
   - 对话日志
   - 数据统计

2. **系统设置**（管理功能）
   - 系统管理（可展开）
     - 用户管理
     - 角色权限
     - API密钥
     - 审计日志
     - 告警规则
     - 系统配置

**菜单项样式**:

| 状态 | 背景色 | 文字颜色 | 特殊标识 |
|------|--------|----------|----------|
| 默认 | 透明 | 浅灰色 | - |
| 悬停 | 白色5%透明 | 白色 | - |
| 激活 | 白色10%透明 | 白色 | 右侧蓝色指示条 |

**交互效果**:
- 圆角卡片样式（rounded-lg）
- 平滑过渡动画（transition-all）
- 图标 + 文字布局
- 子菜单展开/收起动画

### 3. 用户区域（底部）

**位置**: 固定在底部

**显示内容**:
- 用户头像（圆形，带边框）
- 用户名（主要文字）
- 邮箱地址（次要文字）
- 展开图标（右侧）

**下拉菜单**:
- 个人资料
- 账户设置
- 退出登录（红色文字）

**样式**:
```css
背景：悬停时白色5%透明
边框：顶部深色边框分隔
头像：渐变色背景 + 边框
文字：白色主文字 + 灰色副文字
```

## 右侧内容区

### 1. 面包屑导航栏（顶部）

**位置**: 固定在顶部，高度 56px

**功能**:
- 显示当前页面路径
- 支持点击返回上级
- 首页用图标表示

**样式**:
```css
背景：白色
边框：底部灰色边框
文字：灰色路径 + 黑色当前页
分隔符：右箭头图标
```

**示例**:
```
🏠 > Agent管理 > 创建Agent
```

### 2. 主内容区域（中间）

**特点**:
- 浅灰色背景（gray-50）
- 可滚动区域
- 内边距 24px
- 自适应高度

**内容展示**:
- 页面标题和描述
- 功能卡片
- 数据表格
- 表单组件
- 等等...

## 动画效果

### 过渡动画

```css
transition-all duration-300
```

**应用场景**:
- 菜单项悬停
- 菜单项激活
- 子菜单展开/收起
- 按钮交互

### 悬停效果

| 元素 | 效果 |
|------|------|
| 菜单项 | 背景色变化 + 文字颜色变化 |
| 按钮 | 背景色变化 |
| 链接 | 文字颜色变化 |
| 卡片 | 阴影增强 |

## 响应式设计

### 断点

| 设备 | 宽度 | 布局调整 |
|------|------|----------|
| 桌面 | ≥1024px | 完整布局 |
| 平板 | 768-1023px | 侧边栏可折叠 |
| 手机 | <768px | 侧边栏抽屉式 |

### 当前实现

- 固定宽度侧边栏（256px）
- 右侧内容区自适应
- 暂不支持移动端（后续优化）

## 可访问性

### 键盘导航

- Tab 键切换焦点
- Enter 键激活菜单项
- Esc 键关闭下拉菜单

### 语义化HTML

- 使用 `<nav>` 标签包裹导航
- 使用 `<header>` 标签包裹顶栏
- 使用 `<main>` 标签包裹主内容
- 使用 `<aside>` 标签包裹侧边栏

### 对比度

- 文字与背景对比度 ≥ 4.5:1
- 图标与背景对比度 ≥ 3:1

## 最佳实践

### 1. 保持一致性

- 统一的间距系统
- 统一的颜色使用
- 统一的圆角大小
- 统一的动画效果

### 2. 视觉层次

- 使用颜色区分重要性
- 使用大小区分层级
- 使用间距分组内容

### 3. 交互反馈

- 悬停状态明显
- 激活状态清晰
- 加载状态提示
- 错误状态警示

### 4. 性能优化

- 使用 CSS 过渡而非 JS 动画
- 避免过度使用阴影
- 优化图标加载
- 懒加载非关键内容
