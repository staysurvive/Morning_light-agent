# 账户功能说明

## 功能概述

本次更新新增了完整的账户管理功能，包括登录、个人资料和账户设置三个核心页面。

## 1. 登录页面 (`/login`)

### 功能特性
- **邮箱密码登录**：支持邮箱和密码输入
- **验证码功能**：4位随机验证码，支持点击刷新
- **表单验证**：完整的前端验证逻辑
- **Mock登录**：模拟登录流程，保存token和用户信息到localStorage
- **响应式设计**：美观的渐变背景和卡片布局

### 技术实现
- 验证码自动生成（大写字母+数字，排除易混淆字符）
- 点击验证码图片或刷新按钮可重新生成
- 登录成功后跳转到首页
- 使用localStorage存储用户信息和token

### 路由配置
- 路径：`/login`
- 布局：独立页面（不在DashboardLayout内）

## 2. 个人资料页面 (`/profile`)

### 功能特性
- **头像管理**：显示用户头像，编辑模式下可上传新头像
- **基本信息编辑**：
  - 姓名、邮箱、手机号
  - 所在地、部门、职位
  - 个人简介
- **账户统计**：显示创建的Agent、文档、Prompt和调用次数
- **最近活动**：展示最近的操作记录
- **编辑/保存模式**：点击编辑按钮进入编辑模式，可保存或取消

### 技术实现
- 使用useState管理编辑状态
- 头像上传使用FileReader读取本地图片
- 表单字段带图标，提升视觉效果
- 统计数据使用彩色卡片展示

### 路由配置
- 路径：`/profile`
- 布局：在DashboardLayout内
- 入口：左下角用户下拉菜单 → "个人资料"

## 3. 账户设置页面 (`/settings`)

### 功能特性

#### 密码修改
- 当前密码、新密码、确认密码输入
- 密码强度要求提示（至少8位，包含大小写字母、数字和特殊字符）
- 前端验证：密码一致性检查、长度验证

#### 安全设置
- **双因素认证**：开关控制
- **会话超时时间**：可选15分钟、30分钟、1小时、2小时或永不超时
- **API密钥管理**：跳转到API密钥管理页面

#### 通知设置
- 邮件通知
- 浏览器推送通知
- Agent告警通知
- 系统更新通知
- 每周报告

#### 偏好设置
- **语言**：简体中文、繁體中文、English、日本語
- **主题**：浅色、深色、跟随系统
- **时区**：中国标准时间、日本标准时间、美国东部时间、格林威治时间

### 技术实现
- 使用Switch组件控制开关选项
- 使用Select组件选择下拉选项
- 所有设置统一保存按钮
- 支持重置为默认设置

### 路由配置
- 路径：`/settings`
- 布局：在DashboardLayout内
- 入口：左下角用户下拉菜单 → "账户设置"

## 4. 退出登录功能

### 功能特性
- 清除localStorage中的token和用户信息
- 跳转到登录页面

### 入口位置
- 左下角用户下拉菜单 → "退出登录"（红色文字）

## 数据存储

### localStorage存储结构
```javascript
// 登录后存储
localStorage.setItem('token', 'mock-token-xxx')
localStorage.setItem('user', JSON.stringify({
  id: 'user-001',
  name: '管理员',
  email: 'admin@example.com',
  avatar: 'https://api.dicebear.com/7.x/avataaars/svg?seed=admin'
}))
```

## 路由结构

```
/login              - 登录页（独立布局）
/profile            - 个人资料（DashboardLayout内）
/settings           - 账户设置（DashboardLayout内）
```

1. **一致的设计语言**：与整体平台保持一致的白色背景和蓝色主题
